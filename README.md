# Telco_Customer_Prediction
Analyse Predictive : Telco Customer Churn et LTV  Ce projet exploite un jeu de données de plus de 7 000 clients pour traiter deux problématiques. Classification (Variable : Churn) : Identifier les clients susceptibles de résilier leur contrat.    Régression (Variable : Total Charges) : Prédire le montant total facturé.
